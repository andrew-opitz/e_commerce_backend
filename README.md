# e_commerce_backend

## Description
This app was created to serve as the backend for an e-commerce site which are becoming more and more popular and proftable. This app returns data about products, their tags and categories from a simulated e-commerce site database.

## Installation
Node.js, mysql2, sequelize, express and dotenv are required for this application to run.
